# Python-Weather-Aqi
A beginner-friendly Python project that retrieves current weather and air quality (AQI) information for any city using the OpenWeatherMap API.

This project is designed for learners who want hands-on experience with:
-> Working with REST APIs
-> Handling JSON responses
-> Error handling in Python
-> Building small but real-world CLI tools
